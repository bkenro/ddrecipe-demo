# DDRecipe ディストリビューション

## 使い方

```bash
$ git clone https://github.com/bkenro/ddrecipe-demo.git demo
$ cd demo
$ composer install
$ drush @local sql:create -y
$ drush @local site:install ddrecipe -y
```
